** Primitive Data Types: **

* Number - let age = 22;
* String - let firstName = "Ayan";
* Boolean - let adult = true;
* Undefined - Initiated but no value: let hello;
* Null - 'empty value'
* Symbol (ES2015)
* Bigint (ES2020) - Integers that are too big!
